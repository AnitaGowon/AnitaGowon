# Circuit Dataset Analysis
## Objective
Analyze trends in Formula 1 circuits using Python.

## Tools Used
- Python (Pandas, Matplotlib)
- Dataset: circuit.csv

## Key Findings
- Europe hosts 80% of circuits.
- The average circuit altitude is 500m.
